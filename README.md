# Rewriter Pad

A simple application for rewriting files.

## Features

- Graphical user interface built with egui
- Cross-platform support (Windows, Linux, macOS)
- Easy-to-use interface for file operations

## Installation

### Pre-built Binaries

Download the appropriate binary for your platform from the [releases page](https://github.com/m1Ck1-pypy/rewriter-pad/releases).

### From Source

```bash
# Clone the repository
git clone https://github.com/m1Ck1-pypy/rewriter-pad.git
cd rewriter-pad

# Build the application
cargo build --release

# Run the application
cargo run --release
```

## Usage

Run the executable and use the graphical interface to work with your files.

## Development

This project uses Rust and the egui framework for the UI. To contribute:

1. Install Rust from https://rust-lang.org
2. Clone this repository
3. Build with `cargo build --release`
4. Run tests with `cargo test`
